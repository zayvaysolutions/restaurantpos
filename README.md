# RestaurantPOS v3.2

Sistema de Punto de Venta Profesional para Restaurantes.

## Características
- Gestión de ventas con carrito de compras
- Múltiples métodos de pago (Efectivo, Tarjeta, Mixto)
- Reportes de ventas con filtros
- Gestión de inventario con productos eliminados/restaurados
- Dashboard con estadísticas
- Configuración del negocio
- Multi-usuario con roles (admin/cajero)
- Respaldo y restauración de datos
- Impresión de tickets térmicos
- Modo pantalla completa

## Instalación
```bash
npm install
npm start
